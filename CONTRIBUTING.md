# Contributing to RAIA Protocol

## What we welcome

**Jurisdictional modules** — highest-value contribution. Add `/modules/{CC}/README.md` using GB or TH as a template.

**Spec improvements** — clarity, gaps, ambiguities. Open an issue first.

**Schema working group** — JSON schemas publish with v1.0. Open an issue labelled `schema-working-group` to participate.

**SDK implementations** — TypeScript and Python welcome under `/sdk/{language}/`.

## Process

1. Open an issue describing the change and why
2. Maintainer acknowledgement (aim: 5 business days)
3. PR referencing the issue
4. Maintainer review and merge

## Code of conduct

Professional standards body. Constructive, specific, evidence-based contributions only.
