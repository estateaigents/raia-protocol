# TH Jurisdictional Module — Thailand

## Status
Draft. Community contributions welcome.

## Applicable regulations

- Condominium Act B.E. 2522 (foreign ownership 49% quota)
- Foreign Business Act B.E. 2542
- PDPA — Personal Data Protection Act B.E. 2562
- Foreign Exchange Transaction (FET) requirements (Bank of Thailand)

## Accepted compliance bodies

| Body | Type |
|---|---|
| REA Thailand | Professional association |
| FIABCI Thailand | International federation |
| TREBA | Thai Real Estate Broker Association |
| AREA | Agency for Real Estate Affairs |

## Property compliance signals

| Signal | Type | Notes |
|---|---|---|
| foreign_ownership_quota_available | boolean | Condo Act 49% foreign quota |
| fet_gateway_required | boolean | For foreign buyers remitting funds |
| condo_quota_pct_remaining | number | Remaining foreign quota % in building |
