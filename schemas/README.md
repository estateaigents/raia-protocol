# RAIA Protocol Schemas

Formal JSON Schema files publish with v1.0 (planned Q3 2026).

Field definitions are described in prose in [SPEC.md](../SPEC.md).

## Planned schemas

| Schema | Description |
|---|---|
| agent.json | Agent Card — identity, compliance signals, endpoints |
| property.json | Property Card — lettings and sales |
| enquiry.json | Enquiry — full transaction lifecycle |
